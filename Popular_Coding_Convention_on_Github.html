Last comma vs. First comma

  • 93.027 %

    Last comma

  • 6.973 %

    First comma
  • based on 1,501,191 commits

Last comma

var foo = 1,
    bar = 2,
    baz = 3;

var obj = {
    foo: 1,
    bar: 2,
    baz: 3
};

First comma

var foo = 1
  , bar = 2
  , baz = 3;

var obj = {
    foo: 1
  , bar: 2
  , baz: 3
};
